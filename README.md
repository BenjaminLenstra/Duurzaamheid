# Duurzaamheid
Deze kaart bevat de volgende elementen: de te plaatsen elektriciteitshuisjes van Liander ten behoeve van de buurtaanpak voor netverzwaring; bouwjaar en energielabel van de aanwezige huizen en de KLIC aanwezig in het plangebied
De referentieafbeelding van het elektriciteitshuisje is een digitale bewerking van het elektriciteitshuisje van deze link: https://www.liander.nl/regio/gelderland/hoevelaken/stoutenburgerlaan-oost# (fotograaf en datum onbekend).
Data van de Kaartviewer Gemeente Nijkerk en Liander.
